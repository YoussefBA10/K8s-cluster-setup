This repository documents my journey to become an unshakeable DevOps engineer.  
It contains infrastructure scripts, automation, CI/CD pipelines, Helm charts, and documentation.


- `docs/` - Markdown documentation for each module.
- `helm/` - Helm charts for Kubernetes deployments.
- `ansible/` - Playbooks for automation and provisioning.
- `bash/` - Scripts for various DevOps tasks.
- `terraform/` - Infrastructure as Code modules.

Each module includes setup instructions, explanations, and experiments conducted.
